# Визуализатор алгоритмов сортировки

Программа с графическим интерфейсом для пошаговой визуализации и сравнения алгоритмов сортировки.

## Реализованные алгоритмы

В приложении реализованы четыре алгоритма сортировки:

- пузырьковая сортировка;
- сортировка вставками;
- сортировка слиянием;
- быстрая сортировка.

Приложение позволяет наблюдать процесс сортировки в графическом виде. Элементы массива отображаются в виде вертикальных столбцов, высота которых зависит от числового значения элемента.

## Возможности программы

Реализованы основные режимы работы:

- пошаговое выполнение алгоритма;
- автоматическая визуализация сортировки;
- режим «гонки» нескольких алгоритмов на одном исходном массиве;
- подсчет количества сравнений, перестановок, записей и общего числа операций.

Ввод данных выполняется через интерфейс приложения. Пользователь может выбрать размер массива, способ его заполнения и скорость визуализации.

## Способы заполнения массива

Доступные способы заполнения массива:

- случайный массив;
- обратный массив;
- почти отсортированный массив.

Приложение работает на Windows, Linux и macOS.

## Загрузка

Готовые исполняемые файлы для вашей операционной системы можно скачать в разделе **Releases**.

Запуск не требует установки дополнительных программ — достаточно скачать и открыть файл:

- **Windows** — `SortVisualizer.exe`;
- **Linux** — `SortVisualizer-linux`;
- **macOS** — `SortVisualizer-macos`.

Для Linux после скачивания может потребоваться дать файлу права на выполнение:

```bash
chmod +x SortVisualizer-linux
```

## Запуск из исходного кода

Если вы предпочитаете запускать скрипт напрямую, потребуется Python 3.7+ и библиотека `tkinter`. Обычно `tkinter` входит в стандартную поставку Python.

Клонируйте репозиторий:

```bash
git clone https://github.com/ваш_профиль/sort-visualizer.git
cd sort-visualizer
```

Запустите приложение:

```bash
python sort_visualizer_app.py
```

## Описание работы приложения

После запуска программы массив сразу не отображается. Для начала работы необходимо выбрать параметры и нажать кнопку **«Создать массив»**.

В поле **«Размер»** указывается количество элементов массива. Размер должен быть целым числом от 5 до 120. Такое ограничение введено для удобного отображения массива в графическом интерфейсе.

В поле **«Заполнение»** выбирается способ создания массива:

- **«Случайный»** — создается массив случайных целых чисел;
- **«Обратный»** — создается массив, отсортированный в обратном порядке;
- **«Почти отсортированный»** — создается почти упорядоченный массив с несколькими случайными перестановками.

В поле **«Алгоритм»** выбирается метод сортировки. После нажатия кнопки **«Старт»** начинается автоматическая визуализация. Кнопка **«Следующий шаг»** позволяет выполнять сортировку по одному действию. Кнопка **«Пауза»** останавливает автоматическое выполнение, а кнопка **«Сброс»** возвращает сортировку к начальному состоянию.

## Режим «Гонка алгоритмов»

В режиме **«Гонка алгоритмов»** программа создает копии одного исходного массива и запускает все реализованные алгоритмы сортировки. Это позволяет сравнить работу алгоритмов на одинаковых входных данных.

Для каждого алгоритма отдельно отображаются:

- количество сравнений;
- количество перестановок;
- количество записей;
- общее число операций;
- время выполнения.

## Сборка исполняемых файлов

Для сборки приложения используется **PyInstaller**. Автоматическая сборка выполняется через **GitHub Actions**.

Workflow-файл `build.yml` находится в папке `.github/workflows`. Он собирает приложение для Windows, Linux и macOS. После успешной сборки готовые файлы можно скачать как артефакты или опубликовать в разделе **Releases**.

Команда сборки приложения:

```bash
pyinstaller --onefile --windowed --name SortVisualizer sort_visualizer_app.py
```
